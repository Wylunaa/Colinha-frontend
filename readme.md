# 📋 Colinha Frontend

Um guia rápido e interativo para estudantes que estão começando no desenvolvimento **Frontend**.

O projeto reúne **conceitos básicos de HTML, CSS e JavaScript** organizados de forma simples, com exemplos práticos e exercícios para treinar.

A ideia é permitir que qualquer pessoa abra o site, pressione **F12** no navegador e explore o código enquanto aprende.

---

# 🌐 Acesse o projeto

🔗 Em breve disponível via **GitHub Pages**

---

# 🎯 Objetivo do projeto

Este projeto foi criado com fins **educacionais**, para ajudar iniciantes a:

* entender a estrutura básica de um projeto frontend
* aprender conceitos fundamentais de **HTML**
* entender a base do **CSS**
* explorar pequenos comportamentos com **JavaScript**
* praticar através de **exercícios interativos**

---

# 📚 Conteúdos disponíveis

O site contém seções organizadas com:

### HTML

* Estrutura básica do documento
* Tags semânticas
* Elementos de conteúdo
* Navegação e listas

### CSS

* Estilização básica
* Classes e seletores
* Hover e interações simples

### Componentes

* Botões
* Estrutura de layout
* Organização de código

### Exercícios

Desafios divididos em três níveis:

* 📁 HTML
* 🎨 HTML + CSS
* ⚙️ HTML + CSS + JavaScript

Cada exercício permite visualizar o **código da solução**.

---

# 🧠 Como estudar usando o projeto

1. Abra o site no navegador
2. Pressione **F12**
3. Explore o código HTML e CSS
4. Tente resolver os exercícios antes de ver a resposta
